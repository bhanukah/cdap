SRS Report & Presentation

SRS Report Submission Date: 15/05/2018

SRS Presentation Date: 11/06/2018

Accumalated Marks: 6% of final assessment

Deliverables: SRS Document

Before the SRS presentation, it was mandetory to submit a SRS document. System Requirements Specification was created to introduce the concepts of user requirements and system requirements as well as to describe the functional and non functional requirements. Another objective of creating this System Requirements Specification document is to explain how the software requirements can be organized.It was required to create four SRS documents, that means each member in the team should have to submit a document according to his or her research component. The document is mostly focused on, each member's research component. IEEE format was followed by all the members in the document creation.
